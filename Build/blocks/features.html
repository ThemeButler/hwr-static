Features

Easily create dynamic content grids, pulling from content in Headway or WordPress.

Pricing Demo

Add features via Headway

Perfect for quickly adding a feature grid with minimal content. Content is added via the block options in Headway.

Add features via WordPress

For long form content, use the built in Features post type, which can then be filtered and outputted as needed.

Filtering & ordering

Filter your features content based on category, tag, author, or even date. Features can be ordered based on date and post id.

Flexible content grid

Display your features in one column or ten, it's up to you. Also easy to set the spacing between rows and columns.

Feature images or icons

If enabled, each feature can have it's own feature image. Alternatively choose from one of the bundled Font Awesome icons.

Responsive out the box

Features will detect if responsive mode is enabled in Headway and if so, adapt to the available space automatically.
